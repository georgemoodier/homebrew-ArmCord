## homebrew-armcord
The Homebrew repo for installing, uninstalling, and updating ArmCord.

### Getting Started 
Tap the repo:
```
brew tap armcord/armcord
```

install with: 
```
brew install --cask armcord
```

uninstall with:
```
brew uninstall --cask armcord
```
