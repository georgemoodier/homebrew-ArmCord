## homebrew-legcord
The Homebrew repo for installing, uninstalling, and updating LegCord.

### Getting Started 
Tap the repo:
```
brew tap legcord/legcord
```

install with: 
```
brew install --cask legcord
```

uninstall with:
```
brew uninstall --cask legcord
```
